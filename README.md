# Projekt Metrix

Cílem projektu metrix je vytvořit systém pro zobrazování měřících dashboardů jako webovou aplikaci. Cílem projektu
je vytvořit platformu pro různé projekty vyžadující graficky zobrazovat výsledky různých měření.

Projekt metrix poskytuje pouze grafické rozhraní. Měřící backend není součástí projektu.


## Závislosti a instalace

Projekt je založen na projektu [spsostrov-php-runtime](https://github.com/marek-sterzik/spsostrov-php-runtime), pro instalaci se proto řiďte instrukcemi v tomto repozitáři.

## spuštění

```bash
bin/docker start
```
